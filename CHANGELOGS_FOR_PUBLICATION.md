# Changelogs for Publication

This file contains three changelogs ready to be published on the website. Each changelog includes:
- Title
- Excerpt
- Type (changelog)
- Content (HTML-ready)

To publish these changelogs:
1. Go to `/blog/admin/create`
2. Select "Change Log" as the type
3. Copy the title, excerpt, and content from each changelog below
4. Set the published date
5. Click "Publish"
